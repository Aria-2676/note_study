# 项目文档

本目录包含项目的相关文档：

## 文件结构

- **kf.md**：功能模块实现记录
  - 详细记录每个功能的实现方式、核心文件和关键函数

- **architecture.md**：项目架构文档
  - 整体架构描述
  - 模块间依赖关系
  - 数据流说明
  - 状态管理方案
  - 多平台适配

## 文档维护

- **与代码同步**：每次修改功能时同步更新文档
- **版本控制**：文档纳入Git管理
- **定期审查**：确保文档准确性和完整性
- **简洁实用**：只记录关键信息，避免文档膨胀

## 查看指南

1. **功能开发**：参考 `kf.md` 了解现有功能的实现
2. **架构理解**：参考 `architecture.md` 了解项目整体架构
3. **扩展开发**：基于现有架构和功能模式进行扩展
